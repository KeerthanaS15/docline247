# DocLine 247

```sh
# git clone
git clone git@github.com:KeerthanaS15/docline247.git
cd docline247

# set up a virtual environment
mkvirtualenv docline247

# install required packages
pip install -r pip-requirements

# install node_modules
npm install

# install yarn
npm install yarn

# install yarn dependencies
yarn install

# run node-sass to compile scss using yarn
yarn build:css

# watch the changes made to scss files
yarn watch:css

# run migrations
python manage.py migrate

# run dev server
python manage.py runserver
```
